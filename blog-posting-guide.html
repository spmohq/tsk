Blog Posting Guide

Posting Instructions

Step 1: New Blog

1, Title*

2, URL*: /blog/
(Only hyphen, alphabets and numbers allowed)

3, Author*:

4, Category*:
(Multiple categories should be comma separated)

5, SEO Title

6, SEO Keywords
(Multiple keywords should be comma separated)

7, SEO Description

8, Publish Date
(Keep it blank to use Created Date)

"Clear/Submit"

Step 2: Edit Content

(In List of Blog post: on the blog you want to edit):

1, Click Edit

2, Click mouse on body

3, Select More(Mid) > HTML

4, Save or Publish (Top of RHS)

Step 3: Publish Blog

Publish for:

- Members

- Everyone

Step 4: Post-Publish Actions

Go to Dashboard > Membership > Membership Club.

Tap on Say hi.

After you post and it is published select the chat. Tap on the three dots icon beside the emoji icon. And select disable replies.
